[Parameters]
Parameter settings can be entered by editing the URL in the browser:
Insert first parameter after "?" character
Insert multiple parameters using "&" character

Example: http://192.168.0.1/index.html?topmenu=0&zone=1
In this example, the Top Menu button will not be displayed
on the Main control page.

Any combination of the following parameters can be used:
topmenu = 0(no top menu)[default] / 1(On)
zone = 0(TopMenu)[default] / 1(Main) / 2(Zone2) ...
reload = 1(On)[default] / 0(Off)
settings = 1(On)[default] / 0(no settings button)
